# Onpremise Kubeadm

Create a Kubernetes cluster with Kubeadm.

* Red Hat or CentOs only
* CNI : Calico or Flannel
* Load balancer : MetalLB
* VMs must already exists
* VMs DNS/IP must be reachable
* Need SSH access
